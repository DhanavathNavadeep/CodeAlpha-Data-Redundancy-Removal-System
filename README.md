# Data Redundancy Removal System

## Objective

Detect duplicate records before inserting them into the database.

## Features

- Validation of records
- Duplicate detection
- False positive detection
- SQLite database
- Report generation

## Folder Structure

database/
data/
models/
utils/
output/

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Output

database/database.db

output/report.txt
